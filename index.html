01. Add-on Assets

Download the custom skin pack components required for this build.

02. Integration

1. Open the 12.41 Fortnite Folder.
2. Navigate to: FortniteGame > Content > Paks.
3. Paste both downloaded skin pack files into the Paks folder.
4. Close the directory and restart your 12.41 launcher to apply assets.

Security Disclosure

Due to the modification of game pak files, some antivirus software may incorrectly flag this build as a "Trojan" or "Virus." This is a False Positive common with private server assets and custom community content. These files are verified safe for use within Project Wraith.
