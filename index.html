03
Integration

1. Open your extracted 12.41 Fortnite folder.
2. Navigate to FortniteGame › Content › Paks.
3. Move both skin pack files into the Paks directory.
4. Launch the game — assets will load automatically.

Security Disclosure

Due to the modification of game pak files, some antivirus software may incorrectly flag this build. This is a false positive common with private server assets. All files are verified safe for use within Project Wraith.
